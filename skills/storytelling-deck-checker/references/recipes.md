# Storyteller Tactics Recipe Definitions

Reference material for `storytelling-deck-checker`. The main SKILL.md loads this file when it needs beat definitions for recipe selection or audit scoring.

## 1. Man in a Hole
Beats: Comfort Zone → Trigger Event → Crisis / Descent → Turning Point → Recovery → Better Place
Use when: Something went wrong and was fixed, or a threat emerged and was addressed.
Audit emphasis: Descent must be felt; recovery must be earned; ending shows genuine transformation.

## 2. No Easy Way
Beats: Initial Challenge → Early Progress → Setback / Complication → Darkest Moment → Breakthrough → Resolution
Use when: The path was genuinely hard; obstacles and pivots are part of the story.
Audit emphasis: The setback must feel real, not softened; the breakthrough must be earned.

## 3. Rags to Riches
Beats: Overlooked Starting Point → Discovery / Trigger → Struggle for Recognition → Deserved Success
Use when: Positioning something that was underestimated or undervalued.
Audit emphasis: "Rags" must be vivid; transformation must feel deserved, not inevitable.

## 4. Voyage & Return
Beats: Familiar World → Departure into Unknown → Discoveries & Challenges → Return → Changed Perspective
Use when: Research, a pilot programme, a market exploration, or a learning journey.
Audit emphasis: Return must bring back changed knowledge; audience must feel what was learned.

## 5. Pitch Perfect POPP
Beats: Problem (real, felt) → Opportunity (specific, sized) → Practical Steps (credible, concrete) → Promise (vivid Better Place)
Use when: Selling, proposing, fundraising, or making a business case.
Audit emphasis: Problem must be audience-owned; Promise must be emotionally resonant, not just ROI.

## 6. Let Me Explain
Beats: Why This Matters → What It Is → How It Works → What It Means for You → What to Do Next
Use when: Teaching, onboarding, explaining a process, or introducing a framework.
Audit emphasis: "Why this matters" must come before "what it is"; audience benefit must be explicit.

## 7. Campfire Story
Beats: Shared Values → Specific Story Embodying Those Values → Lesson / Principle → Invitation to Carry It Forward
Use when: Internal culture, leadership comms, team narrative, change management.
Audit emphasis: Story must be specific and human; lesson must emerge from story, not precede it.

## 8. Public Speaking Arc
Beats: Hook → Context (why this moment matters) → Core Insight → Evidence & Story → Call to Change / Act
Use when: Keynotes, conference talks, panel sessions, TED-style presentations.
Audit emphasis: Hook must earn attention in 30 seconds; Core Insight must be one memorable sentence.
